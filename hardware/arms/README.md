- Arm base (modified):  
  https://github.com/SIGRobotics-UIUC/LeKiwi/blob/main/3DPrintMeshes/modified_base_arm.stl

- Remaining arm components (SO-101 parts):  
  https://github.com/TheRobotStudio/SO-ARM100/tree/main/STL/SO101
